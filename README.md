# Cybersecurity Portfolio

This repository contains hands-on cybersecurity labs and attack simulations focused on
penetration testing, vulnerability analysis, and security fundamentals.

The projects demonstrate practical experience through controlled lab environments
and Capture The Flag (CTF) challenges.

## Focus Areas
- Web Application Security
- Network Enumeration & Exploitation
- Vulnerability Assessment
- CTF Challenges (TryHackMe, Hack The Box)
-Network attack

## Tools & Technologies
- Kali Linux
- Nmap
- Metasploit
- Burp Suite
- Linux
- Bash & basic Python
